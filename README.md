## 📥 Installation

### 🔧 Step 1: Download Archive
1. Download the `Steam Guard Extensions.rar` archive from GitHub
2. Save the file to a convenient folder on your computer
3. Extract the archive using WinRAR or 7-Zip
4. The extension archive is ready for installation

### 🌐 Step 2: Browser Installation
1. Open Chrome/Edge/Brave and go to `chrome://extensions/`
2. Enable **"Developer mode"** in the top right corner
3. Click **"Load unpacked extension"**
4. Select the extracted folder


### ✅ Step 3: Verification
After installation, you'll see the extension in the list with version 2.1.0 and identifier. The icon will appear in your browser's toolbar.
